# SafePath Guardian AI 🚦

An AI-powered safety and route optimization system designed to help users find safer and smarter paths using geocoding and routing technologies.

## 👨‍💻 Author

**Pankaj Saini (cse24071-ops)**
Student at IIIT Kalyani

## 🚀 Features

* 📍 Destination search with geocoding
* 🗺️ Route optimization using OSRM
* ⚡ Fast frontend built with React + Vite
* 🎨 Clean UI using ShadCN + Tailwind

## 🛠️ Tech Stack

* React (Vite)
* TypeScript
* Tailwind CSS
* OSRM (Open Source Routing Machine)

## 📦 Installation

```bash
git clone https://github.com/YOUR-USERNAME/safepath-guardian-ai.git
cd safepath-guardian-ai
npm install
npm run dev
```
